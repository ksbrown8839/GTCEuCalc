# Generated Pack Assets

`texture-atlas.png`, `texture-atlas.json`, files under `assets/gui/`, and files under `assets/fonts/minecraft/` include or describe visual assets derived from Minecraft and from mods included in GregTech Community Pack Modern.

The code in this repository is licensed separately. These generated data assets are included only for non-commercial, educational, and interoperability use with this planner. Minecraft assets, including GUI textures and bitmap font assets, are owned by Mojang/Microsoft, and mod assets are owned by their respective authors. This project does not claim ownership of those assets.

`assets/fonts/F77MinecraftRegular-0VYv.ttf` is a Minecraft-style webfont used so live browser text can visually match the bitmap font. This project does not claim ownership of that font file.

`assets/gtnh-ui/` contains small UI helper images copied from the GTNH calculator reference project for slot, arrow, and tree-control rendering. This project does not claim ownership of those images.
